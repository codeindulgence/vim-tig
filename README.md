vim-tig
=======
Simple plugin to call tig from inside vim.
It's teh handies!

Installation
------------
Use your most favouritest plugin manager.

Usage
-----
Simply run `:Tig` or bind a key to it, e.g.:
```
map <C-G> :Tig<Cr>
```
